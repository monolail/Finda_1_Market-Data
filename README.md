# 📉 기업 상장폐지 예측 프로젝트 (Finda Market-Data Analysis)
> **2025.12.28 ~ 2026.02.28 | 금융 데이터 분석 및 머신러닝 모델링 텀 프로젝트**

이 프로젝트는 시장 데이터와 재무 지표를 활용하여 기업의 건전성을 평가하고, **상장폐지 가능성을 사전에 예측**하는 인공지능 모델을 구축하는 것을 목표로 합니다. 금융권 데이터 직무 역량을 쌓기 위해 8주간 진행된 체계적인 학습과 실습의 결과물입니다.

---

## 🚀 8주간의 금융 데이터 사이언티스트 여정 (Curriculum)

### **Phase 1: 기반 다지기 (Foundation)**
*   **1주차: 금융 데이터의 이해와 SQL**
    *   금융권 데이터 직무 분석 및 취업 전략 수립
    *   DB 설계 기초 및 SQL 코딩 테스트 대비 (데이터 추출 및 가공 역량 확보)
*   **2주차: Python & AI 에이전트 실습**
    *   제휴카드 가입 예측 모델링 (Binary Classification)
    *   **LLM 기반 RAG(Retrieval-Augmented Generation)** 및 **ReAct 에이전트** 개발 실습 (최신 AI 기술 접목)
*   **3주차: 금융 비즈니스 & 통계**
    *   금융 시스템 아키텍처 및 금융 상품의 구조적 이해
    *   데이터 분석을 위한 기초 통계학 및 금융 도메인 지식 습득

### **Phase 2: 심화 분석 (Deep Dive)**
*   **4주차: 데이터 전처리 및 탐색적 분석(EDA)**
    *   금융 데이터 특화 전처리 기법 실습 및 과제 수행
*   **5~6주차: 리스크 모델링 & 시계열 분석**
    *   **금융 시계열의 특수성:** 비정상성(Non-Stationarity) 처리 기법
    *   **신용 리스크 모델링:** PD(부도확률), LGD(부도시 손실률), EAD(부도시 익스포저) 산출 및 Scorecard 개발
    *   **이상거래탐지(FDS):** Credit Card FDS 모델링 기초
*   **7주차: 실전 데이터 정제 (Data Cleaning)**
    *   실제 금융 데이터(`clean_finance.csv`)를 활용한 결측치, 이상치 처리 및 정밀 정제 실습

### **Phase 3: 프로덕트 & 마케팅 (Product & Marketing)**
*   **8주차: 프로덕트 지표 & 마케팅 애널리틱스**
    *   GA(Google Analytics) 기반 프로덕트 지표 분석
    *   고객 세그먼테이션(Customer Segmentation) 및 마케팅 전략 수립 기반 구축

## 🛠️ Tech Stack & Tools
*   **Language:** Python
*   **Library:** Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
*   **Database:** SQL (MariaDB/MySQL)
*   **AI/LLM:** LangChain (RAG/ReAct Agent 실습)
*   **Analysis:** Jupyter Notebook

---

## 💡 Key Learnings
*   금융 도메인 지식(상품, 시스템, 리스크)을 데이터 분석과 결합하는 **도메인 특화 분석 역량** 확보.
*   단순 모델링을 넘어 **신용 평가(Credit Scoring) 및 리스크 관리** 관점의 데이터 활용법 체득.
*   최신 트렌드인 **LLM 기반 에이전트 기술**을 금융 서비스에 접목해보는 기술적 시야 확장.
