BDA 부트캠프 금융 심화반: FinDA를 신청하게 된 동기와 목표를 작성


저는 다양한 데이터 분석 분야에 관심을 가지고 있으며, 그중 금융 데이터 분석에 특히 깊은 흥미를 느끼고 있습니다. 이를 실무 경험과 포트폴리오로 구체화하고자 FinDA 금융 심화반에 지원하게 되었습니다. 학부 과정에서 통계, 머신러닝, Python 기반 데이터 분석을 학습했지만, 금융 데이터를 다루는 경험은 아직 부족하며, 대부분 개인 프로젝트 중심으로 진행하여 현업에 바로 적용할 수 있는 경험도 제한적이었습니다. 이번 부트캠프를 통해 금융 데이터 수집, 처리, 분석과 머신러닝 모델 활용 능력을 체계적으로 습득하고, 실무 중심 프로젝트를 수행하며 실질적인 역량을 쌓고자 합니다. 또한 다양한 분석 기법과 도메인 지식을 실제 사례에 적용하며 문제 해결 능력을 강화하고, 금융권 데이터 분석 직무에서 전문적이고 전략적인 의사결정을 지원하는 분석가로 성장하는 것을 목표로 하고 있습니다. 이를 통해 금융 데이터 기반 인사이트를 도출하고 의사결정 지원 능력을 갖춘 데이터 분석 전문가로 발전하는 것이 최종 목표입니다.
